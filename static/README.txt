Este directorio debe contener los archivos estáticos del proyecto.

Organizados de la siguiente manera:

- css/    : Archivos CSS específicos del proyecto.
- img/    : Imágenes específicas del proyecto.
- js/     : Archivos JavaScript específicos del proyecto.
- theme/  : (opcional) Archivos de alguna plantilla del proyecto.
- vendor/ : Archivos de dependencias como Bootstrap o JQuery.
